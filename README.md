# AI Photobooth Application

## 📌 Overview
This project is a real-time photobooth application built using OpenCV.

## 🚀 Features
- Face detection
- Live camera feed
- Filters (Grayscale, Blur, Cartoon, Edge)
- Capture and save images

## 🛠 Tech Used
- Python
- OpenCV
- NumPy

## ▶️ How to Run
```bash
pip install opencv-python numpy
python app.py
